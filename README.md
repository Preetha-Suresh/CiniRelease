## CiniRelease

A Flutter mobile application that helps users discover the latest movie releases with ratings, details, and a personalized watchlist.

# Features
- Now Playing Movies — Fetches latest releases using TMDB API
- Ratings & Details — View movie ratings, release date, and overview
- Search Movies — Real-time search functionality
- Detail Page — Click any movie to view full details
- Watchlist — Save movies locally using Hive (offline support)
-  Remove from Watchlist — Easily manage saved movies
# Tech Stack
Flutter — UI Framework
Dart — Programming Language
TMDB API — Movie data
Hive — Local database (for watchlist)
Dio — API calls
